Detailed Description

Square equation solves.

This structure contains solve s of equation . Should be initialized using init_eq_solves function. Rules: if num_of_sol = INF_SOLUTIONS or NO_SOLUTIONS, then x1 and x2 must be NAN if num_of_sol = 1, x1 must be finite, x2 must be nan if num_of_sol = 2, x1 and x2 must be finite
